## Installation
```bash
npm i vue-splash
```

## Usage

Plugin install:

```js
import Vue from 'vue'
import Splash from "vue-splash";

Vue.use(Splash)
```

## Props

| name            | type                             | default    | description                                                            |
| --------------- | -------------------------------- | ---------- | ---------------------------------------------------------------------- |
| source          | String                           | ''         | image source to load your logo                                         |
